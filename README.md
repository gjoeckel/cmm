# Canvas Course Video Manager

Navigator for course media assets (audio, graphics, video sequences, captioned videos) stored in Box. Links open files in Box for download or desktop app editing.

## Source

- **Box folder:** AD-365-V4 (184899190488)
- **Naming:** m1 = module 1, 1-1 = section 1, 1-1-1 = activity 1

## Local development

```bash
cd /Users/a00288946/Projects/canvas_media_manager
npx serve -p 5500
# Open http://localhost:5500
```

## Data

`data/AD-365-V4-map.json` is generated by the Box folder mapping script. To refresh:

```bash
cd /path/to/cursor-ops
source config/box.env && node mcp-box-minimal/scripts/map-box-folder.mjs 184899190488 /Users/a00288946/Projects/canvas_media_manager/data
```
